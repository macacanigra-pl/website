# Development setting

Special setting for ubuntu 24.04lts

1. Install virtual environment library
   `sudo apt install python3-venv`
2. Create an environment
   `python -m venv venv`
3. Activate this env
   `source venv/bin/activate`
4. update pip
   `pip install mkdocs-material[imaging]`
